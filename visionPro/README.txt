
В проекте используется система сборки GULP.
Чтобы установить необходимо открыть командную строку в папке проекта и написать "npm i".
После установки написать "gulp". Откроется страница в браузере.

Для html используется препроцессор Pug.
Для css используется препроцессор Sass.
В папке app находятся файлы pug и sass.

Pug
    Index.pug главный файл для pug. В нем написаны все файлы, которые подключаются к проекту. 
    Mixin.pug файл с миксинами для pug.
    Методология БЭМ.

Sass
    main.sass главный файл для sass. В нем написаны все зависимости. 
    _mixins.sass файл с миксинами для sass.
    _variables.sass файл с переменными для sass.
    Остальные файлы - это файлы проекта, их название и соответствие верстке можно посмотреть в main.sass
    в папке components - стили для разных компонентов(header, footer и т.д.)
    в папке popup - стили для модальных окон(popup)

Gulp
    Компилирует pug в html.
    Компилирует sass в css и минимизирует файл. После сохранения(Ctrl+S) происходят изменения в странице без перезагрузки страницы.
    Используется autoprefixer.

.svg и .jpg файлы оптимизированы при помощи https://tinypng.com/

dist
    В папке dist готовая, скомпилированная верстка.
